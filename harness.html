Agent loop — how the duck harness works

One tool, two nested loops: the solver snapshots the game and asks the agent for a turn; the agent may inspect the board via Python several times, but the turn ends the moment action(...) executes — or when the yield timer expires. Nothing else terminates a game but the clock.
